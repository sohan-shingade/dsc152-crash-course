A researcher models graduate stipend (in $1000s) using GRE score, field of study (Statistics = reference, CS, Data Science), and their interaction.

full_mod <- lm(stipend ~ gre * field, data = grad)

(a) Write the simplified fitted equation for Data Science applicants.
(b) Interpret the coefficient 0.09 (gre:fieldCS) in one complete sentence with correct wording.
(c) A colleague says: "The p-value for gre:fieldDataSci is 0.550, so GRE has the same effect on stipend for Data Science and Statistics students." Is this conclusion valid? Explain.
(a) Fitted equation for Data Science applicants:
Plug in CS = 0, DataSci = 1:
stipend_hat = (-8.20 + 6.30) + (0.46 + (-0.03)) * gre = -1.90 + 0.43 * gre
Key: combine the intercept terms (b0 + b3) and the slope terms (b1 + b5). Show the intermediate step, then simplify.
(b) Interpretation of 0.09:
"For CS applicants, each additional point on the GRE is associated with a predicted stipend that is $90 higher compared to the corresponding increase for Statistics applicants, holding all else constant."
Common trap: Saying "the effect of GRE for CS students is 0.09" — NO. 0.09 is the difference in slopes between CS and the reference (Statistics). The actual slope for CS is 0.46 + 0.09 = 0.55.
(c) Correct answer: No — both reasons apply.

(1) An individual p-value only tests that single coefficient. To test whether GRE's effect is the same across all fields, you need a partial F-test with H0: beta_4 = beta_5 = 0. (2) Even for this single coefficient, p = 0.550 means we fail to reject — we cannot conclude the slopes are equal.

Question 2
Write R Code • Hypothesis Setup

You want to test whether the effect of GRE on stipend differs across the three fields (i.e., whether the interaction is significant).

(a) State the null hypothesis using Greek letters (beta notation).
(b) Write the complete R code: define the null model, the full model, and run the test.
(c) What does the null model represent geometrically (in terms of regression lines)?
(a) Null hypothesis:
H0: β4 = β5 = 0

where β4 is the coefficient on gre:fieldCS and β5 is the coefficient on gre:fieldDataSci. Under H0, the slope of GRE is identical across all three fields.

(b) R code:
null_mod <- lm(stipend ~ gre + field, data = grad) full_mod <- lm(stipend ~ gre * field, data = grad) anova(null_mod, full_mod)
Common trap: Using summary(full_mod) and looking at individual p-values. That tests each interaction term separately, not jointly. Must use anova() for partial F-test.
(c) Answer: Three parallel lines (same slope, different intercepts).

Without interaction terms, the model is stipend = b0 + b1*gre + b2*CS + b3*DataSci. All three groups share slope b1 on GRE, but have different intercepts (b0, b0+b2, b0+b3).

Question 3
Centering & Scaling • Derivation

A researcher fits the following model using centered and scaled predictors:

gre_s <- (grad$gre - mean(grad$gre)) / (2 * sd(grad$gre)) gpa_c <- grad$gpa - mean(grad$gpa) model <- lm(stipend ~ gre_s * gpa_c, data = grad)

The fitted model is:

stipend_hat = 21.5 + 6.8 * gre_s + 2.1 * gpa_c + 1.4 * (gre_s * gpa_c)

Suppose mean(gre) = 150, sd(gre) = 10, mean(gpa) = 3.2.

(a) Interpret the intercept 21.5 in context.
(b) What is the estimated slope of gre_s on stipend for a student with GPA = 3.2? Show your work.
(c) What is the estimated slope of gre_s on stipend for a student with GPA = 4.2? Show your work.
(d) Interpret the coefficient 1.4 in one sentence.
(a) Intercept:
21.5 is the predicted stipend (in $1000s) for a student with the average GRE score (150) and average GPA (3.2).

Because both predictors are centered, when gre_s = 0 and gpa_c = 0, that corresponds to gre = mean(gre) and gpa = mean(gpa).

(b) Slope for GPA = 3.2:
When GPA = 3.2: gpa_c = 3.2 - 3.2 = 0 Slope of gre_s = 6.8 + 1.4 * (0) = 6.8 For a student with average GPA, a 1-unit increase in gre_s (= 2*sd(gre) = 20 points on original GRE) is associated with $6,800 higher predicted stipend.
(c) Slope for GPA = 4.2:
When GPA = 4.2: gpa_c = 4.2 - 3.2 = 1.0 Slope of gre_s = 6.8 + 1.4 * (1.0) = 8.2 For a student with GPA = 4.2, a 1-unit increase in gre_s (= 20 GRE points) is associated with $8,200 higher predicted stipend.
(d) Interpretation of 1.4:
For each 1-point increase in GPA (above the mean), the slope of gre_s on stipend increases by 1.4 ($1,400). In other words, the relationship between GRE and stipend is stronger for students with higher GPAs.
Key insight: The interaction coefficient tells you how the slope of one variable CHANGES as the other variable increases. It is NOT a slope itself.
Question 4
Select All That Apply • Conceptual Traps

Consider a model with interaction: y_hat = b0 + b1*X + b2*Z + b3*X*Z

Select ALL statements that are TRUE:

Correct: A, C, D, E, F

(A) TRUE — Rewrite: y = (b0 + b2*Z) + (b1 + b3*Z)*X. When Z = 0, slope on X is just b1.

(B) FALSE — This is the classic trap. In a model WITHOUT interaction, b1 is "the effect of X holding Z constant." But WITH interaction, the effect of X depends on Z. b1 is only the effect of X when Z = 0 specifically.

This is the #1 most tested trap. "Holding Z constant" wording is ONLY correct in models without interaction (additive models). With interaction, you must specify which value of Z.

(C) TRUE — The slope on X is (b1 + b3*Z). For each 1-unit increase in Z, slope changes by b3.

(D) TRUE — If b3 = 0, slope on X is b1 + 0*Z = b1 regardless of Z. No interaction.

(E) TRUE — Direct from the rearranged model.

(F) TRUE — If Z_c = Z - mean(Z), then Z_c = 0 when Z = mean(Z). Since b1 = slope of X when the Z-variable equals 0, centering makes b1 the slope at the mean of Z.

Question 5
Applied Scenario • Full Exam-Style

A study examines how hours of studying (hours) and whether the student uses practice exams (practice: 0 = no, 1 = yes) relate to exam score. The fitted model is:

score_hat = 45.0 + 3.2 * hours + 5.0 * practice + 1.5 * (hours * practice)
(a) Write the simplified fitted equation for students who DO use practice exams.
(b) A student who does NOT use practice exams studies for 10 hours. What is their predicted score?
(c) Compare the predicted score gain from one additional hour of studying for students who use practice exams vs. those who don't. State both values.
(d) A classmate says: "The coefficient 5.0 on practice means students who use practice exams score 5 points higher on average." Is this correct? If not, when specifically is it true?
(a) Equation for practice exam users (practice = 1):
score_hat = (45.0 + 5.0) + (3.2 + 1.5) * hours = 50.0 + 4.7 * hours
(b) Predicted score (no practice, 10 hours):
score_hat = 45.0 + 3.2(10) + 5.0(0) + 1.5(10)(0) = 45.0 + 32.0 = 77.0
(c) Score gain per additional hour:
Without practice exams (practice=0): slope = 3.2 + 1.5(0) = 3.2 points/hour With practice exams (practice=1): slope = 3.2 + 1.5(1) = 4.7 points/hour
Practice exam users gain 1.5 more points per hour of studying than non-users. This 1.5 is exactly the interaction coefficient.
(d) When is 5.0 the correct gap?
Only when hours = 0. The full difference between practice and no-practice is: 5.0 + 1.5 * hours. At hours = 0, gap = 5.0. At hours = 10, gap = 5.0 + 15.0 = 20.0. The "5 points higher" claim ignores the interaction.
Exam trap: In interaction models, the coefficient on a main effect is its effect only when the other interacting variable = 0. The gap between groups CHANGES depending on the value of the other variable.

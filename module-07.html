Part A Learn: Simple Linear Regression Inference
01 What regression inference means — testing if the slope is real

You already know how to fit a line to data. Regression inference goes further: it asks whether the slope you calculated is real (a genuine relationship in the population) or just noise from your sample.

Fitting a line (DSC 80 stuff)

  • Calculate $\hat{\beta}_0$ and $\hat{\beta}_1$
  • Draw the best-fit line
  • Make predictions

Regression inference (DSC 152)

  • Is $\beta_1 \neq 0$ in the population?
  • Get a p-value for the slope
  • Build a confidence interval for $\beta_1$
The hypothesis test for regression:
$$H_0: \beta_1 = 0 \quad \text{(no linear relationship)}$$ $$H_A: \beta_1 \neq 0 \quad \text{(there IS a linear relationship)}$$ Note: no hats — these are population parameters, not sample estimates.

If we reject $H_0$, we have evidence that X and Y are linearly associated. If we fail to reject, the data are consistent with no linear relationship.

02 The regression equation — what each coefficient means
$$\hat{y} = \hat{\beta}_0 + \hat{\beta}_1 x$$
SymbolNameWhat it represents
$\hat{y}$Predicted valueOur model's prediction for a given $x$
$\hat{\beta}_0$InterceptPredicted $y$ when $x = 0$
$\hat{\beta}_1$SlopePredicted change in $y$ for each 1-unit increase in $x$
Hats matter: $\hat{\beta}_1$ is the sample estimate of the slope. $\beta_1$ (no hat) is the true population slope we're trying to learn about. Hypothesis tests are about $\beta_1$ (no hat). The numbers you compute from data are $\hat{\beta}_1$ (hat).

Example from lecture (March Madness data):

&hat;y = 26.7 + 20.0 · knowledge Intercept: A student with 0 basketball knowledge is predicted to score 26.7 bracket points. Slope: Each additional knowledge point is associated with 20 more bracket points.
03 R code: lm() and summary() — reading the coefficients table

Fitting the model

regression.R
# Fit a simple linear regression
model1 <- lm(bracket_score ~ knowledge, data = march_madness)

# See the full output
summary(model1)

The formula bracket_score ~ knowledge reads as "bracket score explained by knowledge." The outcome (Y) goes on the left of ~, the predictor (X) on the right.

The coefficients table

summary(model1) prints a table with four columns you need to know:

ColumnWhat it isHow to read it
Estimate$\hat{\beta}_0$ or $\hat{\beta}_1$The calculated coefficient value
Std. ErrorSE of the estimateHow uncertain the estimate is
t valueTest statistic= Estimate / Std. Error
Pr(>|t|)p-valueTwo-sided test that the coefficient = 0
Key relationship: t value = Estimate / Std. Error. You can always verify this by hand. The Pr(>|t|) column is the p-value for testing $H_0: \beta = 0$.
Common mistake: Students confuse lm() with t.test(). Use lm() for regression (predicting a numeric Y from a numeric X). Use t.test() for comparing group means.
04 Slope interpretation — "associated with," not "causes"

Interpreting the slope is one of the most common exam questions. There's a specific template you should use.

Slope interpretation template:
"For every 1-unit increase in [X], [Y] is predicted to change by $\hat{\beta}_1$ units, on average."

For observational data, use "is associated with" instead of "causes":
"Every 1-unit increase in [X] is associated with an average difference of $\hat{\beta}_1$ in [Y]."

Wrong wording

"A 1-unit increase in rent causes salary to increase by $28.50."

Can't say "causes" with observational data!

Correct wording

"Every $1 increase in average monthly rent for a 1-bedroom apartment is associated with an average difference in starting salary of $28.50."
When CAN you say "causes"? Only when the data comes from a randomized experiment (Module 6). The ds_jobs dataset is observational — nobody randomly assigned cities different rent prices — so we can only claim association.

05 Intercept interpretation — and when it doesn't make sense
Intercept interpretation template:
"When [X] equals 0, the predicted [Y] is $\hat{\beta}_0$."

The intercept is the predicted Y when X = 0. But that doesn't always make practical sense.

Makes sense

"When knowledge = 0, predicted bracket score is 26.7"
(A person with zero basketball knowledge can still fill out a bracket randomly)

Doesn't make sense

"When height = 0 inches, predicted weight is −150 lbs"
(Nobody has 0 height — this is extrapolation beyond the data)
Exam tip: If the question asks you to interpret the intercept and X = 0 is outside the range of your data, say: "The intercept does not have a meaningful interpretation because $x = 0$ is outside the range of the observed data."

LINE conditions for valid inference

Before trusting p-values from regression, check the LINE conditions:

06 PQ2 Q1a: Interpreting the slope in context

Practice Quiz 2, Question 1a:

A simple linear regression model predicts starting salary (sal) from average monthly rent for a 1-bedroom apartment (rent_1br) using the ds_jobs dataset. The slope estimate is 28.50. Interpret the slope in context.

Answer key (exact wording):
"Every $1 increase in average monthly rent for a 1-bedroom apartment is associated with an average difference in starting salary of $28.50."
Scoring checklist:
1. Says "associated with" (not "causes") — this is observational data
2. Uses units for both X and Y ($1 rent, $28.50 salary)
3. Says "average" or "on average" — it's not an exact prediction
4. Names the actual variables (rent, salary) — not just "X and Y"
07 PQ2 Q1b: Association vs causation in regression

Practice Quiz 2, Question 1b:

A classmate says: "There is statistically significant evidence that living in a more expensive city causes higher starting salaries." Is this correct? Explain.

Answer key (exact wording):
"This statement is incorrect. This is an observational dataset — we did not randomly assign jobs to cities. A significant p-value indicates a statistically significant association, not causation."

Loses points

"We can't say causation because correlation isn't causation."

Too vague — doesn't explain WHY (observational data, confounders).

Full marks

"This is an observational dataset — we did not randomly assign jobs to cities. A significant p-value indicates a statistically significant association, not causation. Confounders such as loc_type, industry type, or other unmeasured variables may be driving both rent prices and salaries."

08 PQ2 Q1c: Residual plots in R

Practice Quiz 2, Question 1c: Create a residual plot to check the equal variance (E) condition.

residual_plot.R
# Step 1: Extract residuals from the model
ds_jobs$resid <- model1$residuals

# Step 2: Plot residuals vs predictor
ggplot(data = ds_jobs, aes(x = rent_1br, y = resid)) +
  geom_point()
What to look for in a residual plot:
• Good: Random scatter around 0, constant spread → equal variance holds
• Bad: Fan shape (spread increases) → unequal variance
• Bad: Curved pattern → linearity assumption violated
Remember the syntax:
• Residuals are stored in model1$residuals
• Plot them against the predictor (X), not the outcome (Y)
• The Y-axis of the residual plot is the residuals, X-axis is the predictor

Q2 — Reading Summary Output

A summary(lm(...)) output shows the following row for a predictor:

Estimate Std.Error t.value Pr(>|t|) age 3.200 0.800 4.000 0.0002

(a) What is the slope estimate? (b) Verify the t value. (c) At $\alpha = 0.05$, is there a significant linear relationship?

(a) $\hat{\beta}_1 = 3.200$
(b) $t = 3.200 / 0.800 = 4.000$ ✓
(c) Yes — $p = 0.0002 < 0.05$, so we reject $H_0: \beta_1 = 0$. There is a statistically significant linear relationship.

This conclusion is not valid. This is an observational dataset — students were not randomly assigned to study a certain number of hours. A significant p-value indicates a statistically significant association, not causation. Confounders such as motivation, intelligence, or course load could be associated with both study hours and GPA.

Part A Learn: Model Diagnostics (LINE Conditions)
01 LINE conditions — the four things that must hold for valid regression inference

Regression gives you coefficient estimates no matter what. But for your p-values, confidence intervals, and hypothesis tests to actually mean anything, four conditions need to hold. They spell LINE.

LetterConditionWhat it means
LLinearityThe true relationship between X and Y is linear (not curved)
IIndependenceObservations are independent of each other
NNormalityThe residuals follow a normal distribution
EEqual varianceThe spread of residuals is constant across all X values (homoscedasticity)
ALL four must hold for regression inference to be valid. If any one is violated, your p-values and CIs could be wrong.
The model itself still runs. R will happily give you coefficients even if every condition is violated. The issue is whether you can trust the statistical inference (p-values, CIs, F-tests) built on top of those coefficients.
02 Linearity — residuals vs fitted should show NO pattern

What to check: The relationship between X and Y should be a straight line, not a curve. If you fit a line to curved data, your predictions are systematically wrong.

How to check: Plot residuals vs. fitted values (or residuals vs. the predictor). You want to see random scatter around zero — no curves, no patterns.

check_linearity.R
# Base R diagnostic plots
plot(model, which = 1)  # residuals vs fitted

# Or manually with ggplot
df$resid <- model$residuals
df$fitted <- model$fitted.values
ggplot(data = df, aes(x = fitted, y = resid)) +
  geom_point() +
  geom_hline(yintercept = 0, linetype = "dashed")

Linearity holds

Points scattered randomly around zero. No obvious curve or pattern. Looks like a shapeless cloud.

Linearity violated

Clear U-shape or curve in the residual plot. Points systematically above zero, then below, then above again.
Why not just look at the original scatter plot? For multiple regression (more than one predictor), you can't see the overall relationship in a single scatter plot. Residual plots reveal patterns that are hidden in the raw data.
03 Independence — think about the study design

What to check: Each observation should be independent of all others. One data point shouldn't give you information about another.

How to check: This one is mostly about thinking about how the data was collected, not about a plot.

Common violations of independence:
  • Time series data — today's stock price is related to yesterday's
  • Clustered data — students in the same class, patients at the same hospital
  • Repeated measures — measuring the same person multiple times

If the data is collected over time, you can plot residuals in order of collection to look for trends:

check_independence.R
# Plot residuals in order of data collection
plot(model$residuals, type = "l")
abline(h = 0, lty = 2)
On the exam: Independence is the hardest condition to check from a plot alone. The question usually asks you to think about the data source. If they say "data was collected monthly" or "repeated observations on the same subjects," that's a hint toward an independence violation.
04 Normality — QQ plot should follow the diagonal

What to check: The residuals ($\varepsilon_i = y_i - \hat{y}_i$) should be approximately normally distributed.

How to check: A QQ (quantile-quantile) plot. If the residuals are normal, the points follow the diagonal line closely.

check_normality.R
# Base R QQ plot
plot(model, which = 2)

# Or manually
qqnorm(model$residuals)
qqline(model$residuals)

Normality holds

Points hug the diagonal line closely. Minor wiggles at the tails are normal (pun intended) and fine.

Normality violated

Points curve away from the line systematically, especially at the tails. S-shapes or banana shapes are bad signs.
Good news: With large sample sizes, the Central Limit Theorem makes the normality condition less important. The sampling distribution of $\hat{\beta}$ will be approximately normal even if individual residuals aren't perfectly normal. So mild violations in large samples are usually okay.
05 Equal variance — no fan or funnel in the residual plot

What to check: The spread (variance) of residuals should be roughly the same for all values of X. The fancy word is homoscedasticity.

How to check: Same residuals vs. fitted plot as linearity. But now you're looking at the vertical spread, not the shape.

check_equal_var.R
# From PQ2 Q1c: residual plot to check equal variance
df$resid <- model$residuals
ggplot(data = df, aes(x = predictor, y = resid)) +
  geom_point()

Equal variance holds

Residuals form a band of roughly constant width across all X values. Like a horizontal rectangle of points.

Equal variance violated

Fan/funnel shape — residuals spread out (or narrow) as X increases. Like a megaphone or cone. This is called heteroscedasticity.
Same plot, two checks: The residuals vs. fitted plot checks BOTH linearity (look for curves) AND equal variance (look for changing spread). One plot, two conditions.
Classic exam scenario: "Here is a residual plot. Which LINE condition does it check?" If it's residuals vs. fitted values → could be checking Linearity or Equal variance. Read the question carefully to see which one they're asking about.
Part B On the Exam: Model Diagnostics Questions
06 Exam Q Type: "Write R code for a residual plot"

Real exam question (PQ2 Q1c):

You have a model called model. Create a column of residuals in the data frame, then plot them against a predictor to check the equal variance condition.

Answer key (PQ2 Q1c):
resid_plot_pq2.R
# Step 1: Extract residuals
df$resid <- model$residuals

# Step 2: Plot residuals vs predictor
ggplot(data = df, aes(x = predictor, y = resid)) +
  geom_point()
Key R syntax to memorize:
• model$residuals — extracts the residuals from a fitted model
• model$fitted.values — extracts the predicted $\hat{y}$ values
• You can plot residuals against a predictor OR against fitted values — both work for checking L and E
Your Turn

You have a model house_mod <- lm(price ~ sqft, data = houses). Write R code to add a residuals column to houses and create a ggplot of residuals vs. sqft.

houses$resid <- house_mod$residuals
ggplot(data = houses, aes(x = sqft, y = resid)) +
  geom_point()
07 Exam Q Type: "Which LINE condition does this plot check?"

The exam loves showing you a plot and asking which condition it checks. Here's your cheat sheet:

Plot typeChecks which condition(s)What to look for
Residuals vs. fitted valuesLinearity + Equal varianceNo curve (L), constant spread (E)
Residuals vs. predictorLinearity + Equal varianceSame as above
QQ plot of residualsNormalityPoints follow the diagonal line
Residuals vs. order / timeIndependenceNo trend or pattern over time
Histogram of residualsNormalityBell-shaped distribution
Common trick question: They show a residuals vs. fitted plot with a clear funnel shape and ask "which LINE condition is violated?" Answer: Equal variance (E), not linearity. A funnel = changing spread, not a curve.
Another trick: "No plot can check Independence directly." If the question asks about independence, the answer usually involves thinking about the study design (time series? repeated measures?), not looking at a plot.
Your Turn

Match each scenario to the LINE condition being checked or violated:

(a) A QQ plot shows points curving away from the diagonal at both tails
(b) A residuals vs. fitted plot shows a clear U-shaped curve
(c) A residuals vs. fitted plot shows points fanning out to the right
(d) Data was collected as monthly averages over 5 years

(a) Normality — QQ plot checks N; tails curving away = non-normal residuals
(b) Linearity — U-shaped curve = nonlinear relationship
(c) Equal variance — fanning out = heteroscedasticity
(d) Independence — monthly time series data = observations likely correlated over time
08 Exam Q Type: "What happens when LINE conditions are violated?"

This is the "so what?" question. Why do we care if a condition is violated?

The bottom line: When LINE conditions are violated, you cannot trust the inference. Specifically:
  • p-values become unreliable (could be too small or too large)
  • Confidence intervals may not have the claimed coverage (a "95% CI" might only cover 80% of the time)
  • Hypothesis tests can give misleading results

Loses points

"The model is wrong."

Too vague. Need to specify WHAT goes wrong: inference tools become unreliable.

Full marks

"If LINE conditions are violated, the p-values and confidence intervals from the regression output are unreliable. We cannot trust the results of hypothesis tests or use the standard errors to construct valid intervals."
Important nuance: The coefficient estimates ($\hat{\beta}$) themselves can still be reasonable even when conditions are violated. It's the inference built on top — the standard errors, p-values, and CIs — that falls apart. You can still describe the fitted line, just can't make probabilistic claims about it.
Your Turn

A student fits a regression model and gets a p-value of 0.03. However, the residual plot shows a clear funnel shape. Should the student trust this p-value? Why or why not?

No. The funnel shape in the residual plot indicates a violation of the Equal variance condition. When this condition is violated, p-values are unreliable — the true p-value could be larger or smaller than 0.03. The student cannot trust this result for making inference.
Part C Quiz: Model Diagnostics (LINE)
09 4-Question Quiz
Q1

Name all four LINE conditions and briefly describe what each one requires.

L — Linearity: The true relationship between X and Y is linear (no curves).
I — Independence: Observations are independent of one another (no clustering, no time dependence).
N — Normality: The residuals are approximately normally distributed.
E — Equal variance: The spread of residuals is constant across all values of X (homoscedasticity).
Q2

Match each diagnostic plot to the LINE condition it primarily checks:

(a) QQ plot of residuals
(b) Residuals vs. fitted values showing a funnel shape
(c) Residuals vs. fitted values showing a U-shaped curve

(a) Normality (N) — QQ plot checks whether residuals are normally distributed
(b) Equal variance (E) — funnel/fan shape means spread is changing (heteroscedasticity)
(c) Linearity (L) — U-shape means the true relationship is curved, not linear
Q3

You fit fit <- lm(mpg ~ weight, data = cars). Write R code to: (1) add a residuals column to cars, and (2) create a residual plot using ggplot to check the equal variance condition.

cars$resid <- fit$residuals
ggplot(data = cars, aes(x = weight, y = resid)) +
  geom_point()
Q4

A regression model's residual plot shows clear heteroscedasticity (funnel shape). The model reports a slope p-value of 0.04. A student says: "Since the p-value is below 0.05, the relationship is statistically significant." Do you agree? Explain.

No. The funnel shape indicates a violation of the Equal variance (E) condition. When LINE conditions are violated, p-values are unreliable — the reported p-value of 0.04 may not reflect the true significance level. We cannot trust this p-value to draw conclusions about statistical significance. The confidence intervals from this model would also be unreliable.
